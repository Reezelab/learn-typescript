## Petunjuk Installasi
Silahkan ikuti urutan penggunaan perintah dibawah ini

```bash
npm init -y
```

```bash
npm install --save-dev ts-node typescript 
```

```bash
npx tsc --init
```

```bash
npx ts-node src/index.ts
```

Gunakan perintah tsc untuk mengcompile ke JS
```bash
npx tsc 001_Hello_World/hello_world.ts
```
